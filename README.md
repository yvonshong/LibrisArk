## 📖 About The Project

**LibrisArk** is designed for modern researchers. It is a local-first, AI-enhanced desktop literature management tool that solves the pain points of traditional reference managers (such as Zotero and Mendeley) regarding seamless AI integration, flexible local file management, and strict privacy protection. 

Our goal is to provide an exceptionally fluid reading experience combined with intelligent knowledge extraction, allowing you to focus on discovering insights rather than organizing files.

### ✨ Key Features

- **Local-First Architecture:** Built with Tauri (Rust) and SQLite, ensuring high performance, low memory footprint, and absolute control over your local data.
- **AI Copilot Integration:** Instantly generate structured summaries and extract smart tags for any PDF. Chat directly with your papers using a context-aware AI session.
- **Cost & Privacy Optimized:** AI chat history is persisted locally. This means no redundant tokens are spent re-reading the same PDF across multiple sessions.
- **Modern & Dynamic UI:** A beautiful, responsive interface built with React, TailwindCSS, and Shadcn/UI. Features include a sleek icon-only sidebar, dynamic tag filtering, and built-in PDF reader with highlighting support.
- **Cross-Platform:** Available for Windows, macOS, and Linux.

  <h2>📖 关于本项目 (About The Project - 中文)</h2>
</div>

**LibrisArk** 是一款专为现代科研人员量身打造的**本地优先 (Local-First)**、**AI 增强 (AI-Enhanced)** 的桌面端文献管理工具。

它的诞生旨在解决传统文献管理软件（如 Zotero, Mendeley）在 AI 深度整合、本地文件灵活管理以及数据隐私保护方面存在的痛点。LibrisArk 致力于为您提供极为流畅的阅读体验和智能化的知识提取功能，让您的精力回归到科研洞察本身，而非繁杂的文件整理上。

### ✨ 核心功能

- **本地优先架构:** 采用 Tauri (Rust) + SQLite 构建，保证了极致的启动性能、极低的内存占用，同时让您对本地科研数据拥有绝对的控制权。
- **AI Copilot (智能副驾):** 一键为 PDF 文献生成结构化总结和智能标签。您可以在侧边栏直接与当前论文对话，进行深入探讨。
- **省流与隐私保护:** AI 对话的上下文和历史记录会自动保存在本地数据库中。当您再次打开同一篇文献时，无需让 AI 重新阅读整个 PDF，有效节省 Token 消耗，且保护隐私。
- **现代化动态界面:** 使用 React, TailwindCSS 和 Shadcn/UI 构建的精美响应式 UI。拥有极简的图标侧边栏、丝滑的标签筛选器、以及原生支持高亮批注的内置 PDF 阅读器。
- **跨平台支持:** 完美运行于 Windows, macOS 以及 Linux。
